# web-logger

Place all files from `html` folder into `/var/www/html`.

To make it more secure agaist attackers you can modify apache2 default configuration (/etc/apache2/sites-available/000-default.conf) using `000-default.conf` from this repository.

Don't forget to modify the configuration inside `index.php` file!
